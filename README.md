## 🎯 Objetivo do Projeto
Este projeto realiza uma Análise Exploratória de Dados (EDA) utilizando a biblioteca Pandas sobre a base de dados histórica de homicídios intencionais disponibilizada pelo UNODC (ONU). O objetivo é responder a um conjunto de 10 perguntas analíticas estruturadas e aplicar um modelo de Regressão Linear para projetar a tendência de homicídios globais para os anos de 2023, 2024, 2025 e 2026.

## 📂 Estrutura do Repositório
* `Análise_Homicídios.ipynb`: Notebook contendo o carregamento, tratamento dos dados, resolução das questões e o modelo preditivo.
* `data_cts_intentional_homicide.csv`: Base de dados oficial utilizada na análise.

## 🚀 Como Executar
O notebook principal pode ser visualizado e executado diretamente no Google Colab através do link de atalho presente dentro do arquivo `.ipynb` deste repositório.
